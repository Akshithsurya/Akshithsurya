I'm in grade 10 in Palakkad, Kerala — the district the Kerala School of Mathematics called home six centuries before me. That's not trivia, it's the frame I build in: old mathematical tradition, new hardware.

Two things I keep coming back to: reading light curves for signs of planets that shouldn't be visible, and taking apart ESP32s to find out what they'll do if you ask nicely with the wrong protocol.

---

### 🛰️ What I'm building

- **[EXOVision](https://github.com/Akshithsurya)** — exoplanet detection from transit photometry, ML-driven. Went back through my own repo and fixed what I'd shipped broken the first time: an exponential-smoothing forecast that silently failed, a missing `calculateHabitability()` method, a dead API key fallback, a duplicated source file. The unglamorous kind of progress.

- **ESP_Penetrator** — an ESP32 hardware pentesting toolkit. Core active project.

- **Vulcan** — a Bash-based security research framework. Same category: build the tool, then use the tool to find out where the tool is wrong.

- **IoT Firmware Collection** — an ongoing set of firmware for the hardware I keep collecting excuses to buy.

- **Arivu** (arivu-pallakad.netlify.app) — bilingual Malayalam/English platform for science career literacy, built with Aadi Mahesh. This is the one meant to outlast me: not a personal showcase, a resource for the next kid in Palakkad who doesn't know astrophysics is a job you can have.

*Mentored by Dr. Aswin Sekhar, planetary scientist — which mostly means someone periodically tells me which of my ideas are actually testable.*
